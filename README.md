# LINE Bot with Gemini API

这是一个使用Google Gemini API的LINE Bot。

## 安装

1. 克隆仓库：
   ```
   git clone https://github.com/yourusername/linebot_chatgpt.git
   cd linebot_chatgpt
   ```

2. 安装依赖：
   ```
   npm install
   ```

3. 复制环境变量示例文件并填写实际值：
   ```
   cp .env.example .env
   ```

4. 在`.env`文件中填写你的LINE和Google Gemini API密钥
